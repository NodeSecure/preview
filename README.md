## Contributing

First, install dependencies

```bash
$ npm i
```

Run the development server:

```bash
$ npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Run e2e test:

- First, install playwright.

```bash
$ npx playwright install
```

- Then, run the tests ^^

```bash
$ npm run test:e2e
```
